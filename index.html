📖 시스템 사용법

🎨 테마 & 접근성

  • Light 테마: 기본 브랜드 색상 중심의 밝은 인터페이스
  • Dark 테마: 접근성 목적의 어두운 Dark 테마 인터페이스
  • 큰글씨 모드: 모든 텍스트가 1.2배 확대 (40px → 48px)
  • Static 요소: 포커스/호버는 테마 독립적 핑크 색상
  • 단축키: Ctrl+Alt+H (테마), Ctrl+Alt+L (큰글씨)
  • 설정이 자동으로 저장되어 다음 방문 시에도 유지됩니다

🔘 버튼 타입

  • 기본 버튼: 기본 상태 ↔ 눌림 상태
  • 토글 버튼: 기본 상태 ↔ 선택 상태
  • 비활성 버튼: 비활성 상태 (상호작용 불가)

🎯 버튼 상태

  • 기본 상태: 버튼의 기본 모습
  • 눌림 상태: 기본 버튼이 눌린 상태
  • 선택 상태: 토글 버튼이 선택된 상태
  • 비활성 상태: 상호작용이 불가능한 상태
  • 초점 상태: 모든 버튼에 복합적으로 적용
  • 호버 상태: 모든 버튼에 복합적으로 적용

⌨️ 키보드 조작

  • Tab: 버튼 간 초점 이동
  • Enter 또는 Space: 버튼 활성화
  • Ctrl+Alt+H: 테마 전환
  • Ctrl+Alt+L: 큰글씨 모드 전환

🖱️ 마우스 조작

  • 클릭: 기본 버튼은 눌림 상태, 토글 버튼은 선택 상태 토글
  • 호버: 마우스 오버 시 호버 상태 표시

접근성 정보

이 데모는 웹 접근성 가이드라인(WCAG 2.1 AA)을 준수합니다:

  • 키보드 전용 네비게이션 지원
  • 스크린 리더 호환성
  • 충분한 색상 대비 (4.5:1 이상)
  • 현대 CSS 기능 활용 (Nesting, Container Queries, color-mix)
