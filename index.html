📖 시스템 사용법

🎨 테마 & 접근성

  • Light 테마: 기본 브랜드 색상 중심의 밝은 인터페이스
  • Dark 테마: 접근성 목적의 어두운 Dark 테마 인터페이스
  • 큰글씨 모드: 모든 텍스트가 1.2배 확대 (40px → 48px)
  • Static 요소: 초점 상태는 테마 독립적 핑크 색상
  • 단축키: Ctrl+Alt+H (테마), Ctrl+Alt+L (큰글씨)
  • 설정이 자동으로 저장되어 다음 방문 시에도 유지됩니다

🔘 버튼 타입

  • 기본 버튼: 기본 상태 ↔ 눌림 상태 (.pressed)
  • 토글 버튼: 기본 상태 ↔ 눌림 상태 (.pressed.toggle)
  • 비활성 버튼: 비활성 상태 (상호작용 불가)

🎯 버튼 상태

  • 기본 상태: 버튼의 기본 모습
  • 눌림 상태: 기본 버튼은 일시적, 토글 버튼은 지속적 눌림
  • 비활성 상태: 상호작용이 불가능한 상태
  • 초점 상태: 모든 버튼에 복합적으로 적용 (hover + focus)

⌨️ 키보드 조작

  • Tab: 버튼 간 초점 이동
  • Enter 또는 Space: 버튼 활성화
  • Ctrl+Alt+H: 테마 전환
  • Ctrl+Alt+L: 큰글씨 모드 전환

🖱️ 마우스 조작

  • 클릭: 기본 버튼은 일시적 눌림, 토글 버튼은 지속적 눌림 토글
  • 초점: 마우스 오버 또는 키보드 포커스 시 초점 상태 표시

🎨 커스텀 팔레트 테스트

사이드바에서 생성한 커스텀 팔레트를 실시간으로 테스트할 수 있는 버튼들입니다.

Primary1 팔레트 (Light 테마 기본)

Light 테마의 기본 브랜드 색상을 사용하는 버튼들입니다. Dark 테마에서는 Dark 테마 색상으로 자동 전환됩니다.

Primary2 팔레트 (Light 테마 변형)

Light 테마의 변형 색상을 사용하는 버튼들입니다. Gray 색상과 조합하여 다양한 대비를 제공합니다.

Secondary1 팔레트 (Gray 기반)

테마에 독립적인 Gray 색상을 기반으로 하는 보조 버튼들입니다. 모든 테마에서 일관된 색상을 유지합니다.

Secondary2 팔레트 (Gray 변형)

Gray 색상의 변형을 사용하는 보조 버튼들입니다. 다양한 회색조로 계층적 인터페이스를 구현합니다.
