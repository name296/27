버튼 컴포넌트 시스템

다양한 상태와 팔레트를 지원하는 접근성 중심의 버튼 컴포넌트입니다.

🎨 테마 시스템 사용법:
  • Light 테마: 기본 브랜드 색상 중심의 밝은 인터페이스
  • Dark 테마: 고대비 모드, 접근성 최적화된 어두운 인터페이스
  • Static 요소: 포커스/호버는 테마 독립적 핑크 색상
  • 버튼 클릭 또는 Ctrl+Alt+H 단축키로 전환
  • 설정이 자동으로 저장되어 다음 방문 시에도 유지됩니다
  • 시스템 접근성 설정과 자동으로 연동됩니다

Primary-1 팔레트 (Light 테마 기본)

Light 테마의 기본 브랜드 색상을 사용하는 버튼들입니다. Dark 테마에서는 고대비 색상으로 자동 전환됩니다.

Light 테마의 변형 색상을 사용하는 버튼들입니다. Neutral 색상과 조합하여 다양한 대비를 제공합니다.

Neutral 색상의 변형을 사용하는 보조 버튼들입니다. 다양한 회색조로 계층적 인터페이스를 구현합니다.

사용법

버튼 타입

  • 기본 버튼: 기본 상태 ↔ 눌림 상태
  • 토글 버튼: 기본 상태 ↔ 선택 상태
  • 비활성 버튼: 비활성 상태 (상호작용 불가)

버튼 상태

  • 기본 상태: 버튼의 기본 모습
  • 눌림 상태: 기본 버튼이 눌린 상태
  • 선택 상태: 토글 버튼이 선택된 상태
  • 비활성 상태: 상호작용이 불가능한 상태
  • 초점 상태: 모든 버튼에 복합적으로 적용
  • 호버 상태: 모든 버튼에 복합적으로 적용

키보드 조작

  • Tab: 버튼 간 초점 이동
  • Enter 또는 Space: 버튼 활성화

마우스 조작

  • 클릭: 기본 버튼은 눌림 상태, 토글 버튼은 선택 상태 토글
  • 호버: 마우스 오버 시 호버 상태 표시
